plugmap.com
===========

A site for finding power outlets.

## Config variables

- S3_ACCESS_KEY
- S3_SECRET_KEY
- S3_BUCKET_NAME
- S3_ENDPOINT
- MAPBOX_TILES: The ID of the tiles used by MapBox (passed to the JS API constructor).

## Setup

1. Set up database, DNS, S3 bucket, set corresponding config variables.
